# RAPCAT
 Robust-point-matching- And Piecewise-affine-based Cell Annotation Tool for Caenorhabditis elegans L1 cell annotation


# Usage
Once downloaded, go to `RAPCAT-main` folder, and run `RAPCAT('exampleData/')` for example Data annotation.
After finished, you can check the result with vano program in `vano_win32_1.741` folder.
To use vano for worm visualization, open `wano.exe`, `Press No` for stack image as rows, and than drag the `nhr-206_XIL0604_20140708_0001_S1.ano` or `nhr-206_XIL0604_20140708_0001_S1_recog.ano` file to vano interface.
